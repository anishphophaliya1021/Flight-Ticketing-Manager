=======================  README  ==============================
This is project models a basic flight ticketing system for a single airline. All the flight information is stored in a file flight.txt.
The text file is formatted such that each line is tab delimitted and contains a flight number, number of smoking seats and the number of non-smoking seats that are already reserved.
This is a version 1.0, future version will include more features such as login id, multiple airlines, threaded access to the database.
